Prerequisites

Mockmach, at it's core, is a Jekyll application. Jekyll serves as the engine that generates the raw HTML for your project.

Required for MockMach
  1. Jekyll to generate static pages.
  2. Bower for ease of package management.

Setup

The full installation process is documented and maintained on Mockmach repository.
