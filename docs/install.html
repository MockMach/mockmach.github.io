Prerequisites

Mockmach leverages a few web tools like Bower and Gulp and the static HTML is generated using Jekyll. You'll need to install all the following prerequisites to develop using Mockmach.

Required for MockMach
  1. NodeJS
  2. Jekyll

Setup

The full installation process is documented and maintained on Mockmach repository.
