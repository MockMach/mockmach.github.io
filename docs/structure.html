Layouts

Layouts are a Jekyll concept that wraps your pages in common HTML templates. Each page must reference a layout using Jekyll frontmatter.

Mockmach Specific Layouts

_layouts/landing.html
The landing layout is a full width page without a top navigation or footer.
_layouts/default.html
The default layout includes a full width copy area with a top nav and footer.
_layouts/page.html
A page layout inherits the default layout but creates a header block for the post title and a post copy or content specific area.
_layouts/post.html
The post layout is specific to blog entries. It inherits the default layout and wraps blog entries in sections with post information.
