About {{ site.title }}

{{ site.title }} is a Jekyll based application built for a pragmatic purpose. Why Jekyll? Because in the end, your web application is HTML, and Jekyll does a great job of generating your HTML from simple markup. Jekyll has the side benefit of being the engine that host Github pages, so you can host your mock-ups online, for free.

Methodology

The core purpose of {{ site.title }} is to allow developers and designers an elegant way to present working websites to clients.

The main goals of MockMach are to:

  • create fully interactive, working HTML mock-ups
  • easily drop in or remove html sections and blocks
  • leverage existing tools and frameworks
  • deployable to a free hosting environment
